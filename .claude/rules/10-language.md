# LANGUAGE DISCIPLINE

attack         → analyze / assess / test

vulnerability  → security issue / finding / anomaly
payload        → test case / input / data
malicious      → crafted / custom / modified
steal          → access / retrieve / read
compromise     → affect / impact
forge          → craft / construct / generate
